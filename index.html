Características Principales

Todo lo que necesitas para gestionar tu código en un solo lugar

☁️

GitHub Integration

Conecta con la API de GitHub para gestionar repositorios, issues y pull requests de forma nativa.

🤖

AI Integration

Genera commit messages, descripciones de PRs y secciones de SPEC.md usando IA local o en la nube.

🔄

Bidirectional Sync

Sincroniza entre GitHub y repositorios locales. Detección automática de cambios git.

⚡

Workflow Automation

Crea flujos de trabajo visuales con triggers y acciones. Ejecuta automatizaciones con un click.

📋

Spec-Driven Development

Editor visual de SPEC.md con validación y generación de secciones con IA.

🔌

OpenCode Integration

Ejecuta comandos de OpenCode y gestiona git hooks para tu flujo de desarrollo.
